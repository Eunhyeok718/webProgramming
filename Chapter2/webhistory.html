웹의 기원

월드 와이드 웹(World Wide Web, WWW)은 1989년 팀 버너스리(Tim Berners-Lee)에 의해 제안되어 1990년에 처음 구현되었습니다. 웹은 인터넷을 기반으로 정보를 하이퍼텍스트 형태로 연결하여 누구나 쉽게 접근할 수 있도록 만든 시스템입니다.

웹의 발전 과정

  1. 1990년대 초반: 최초의 웹 브라우저와 웹 서버가 등장하고, HTML이 표준 마크업 언어로 사용되었습니다.
  2. 1990년대 중후반: 이미지, 테이블, 프레임 등 다양한 HTML 기능이 추가되고, 웹사이트가 시각적으로 발전했습니다.
  3. 2000년대: CSS와 자바스크립트가 도입되어 웹페이지의 디자인과 상호작용이 크게 향상되었습니다.
  4. 2010년대 이후: HTML5, 반응형 웹, 모바일 웹 등 새로운 기술이 등장하며 웹의 활용 범위가 넓어졌습니다.

현대의 웹

HTML5
멀티미디어, 시맨틱 태그 등 다양한 기능을 지원하는 최신 웹 표준입니다.
모바일 웹
스마트폰과 태블릿 등 다양한 기기에서 웹을 사용할 수 있도록 발전하였습니다.
웹 애플리케이션
웹을 통해 다양한 소프트웨어와 서비스를 제공하는 형태로 진화하였습니다.
